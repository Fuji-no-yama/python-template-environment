# コーディングルール
- コードの変更があった場合は最後に必ず`uv run ruff check`, `uv run ty check`を実行してエラー・警告がないかを確認する
- src以下のコードについて、公開関数(_が先頭につかないもの)についてはすべてgoogl式でdocstringをつけること(日本語・以下の形式)。一方で、@function_toolがついている関数については同様のdocstringを英語で作成すること
```python
def func(a:int, b:int) -> int:
    """
    aとbを加算して返す。

    Args:
        a (int): 加算する整数1つ目
        b (int): 加算する整数2つ目

    Returns:
        int: 加算結果
    """
```
- commitメッセージは「prefix付きで英語」をデフォルトとする
  ┌────────────────┬──────────────────────────────────────┐                                                                                                                                
  │ プレフィックス │                 用途                 │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ feat:          │ 新機能の追加                         │
  ├────────────────┼──────────────────────────────────────┤
  │ fix:           │ バグ修正                             │
  ├────────────────┼──────────────────────────────────────┤
  │ docs:          │ ドキュメントの変更                   │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ style:         │ フォーマット変更（動作に影響しない） │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ refactor:      │ リファクタリング                     │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ test:          │ テストの追加・修正                   │
  ├────────────────┼──────────────────────────────────────┤
  │ chore:         │ ビルド・ツール・設定などの雑務       │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ perf:          │ パフォーマンス改善                     │
  ├────────────────┼──────────────────────────────────────┤                                                                                                                                
  │ ci:            │ CI関連の変更                           │
  └────────────────┴──────────────────────────────────────┘
- hasattrは基本使わない。どうしても使う場合は確認を取る。
